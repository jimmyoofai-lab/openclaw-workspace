# ONBOARDING.md — Авторизация новых пользователей

## Как определить нового пользователя

При каждом сообщении проверяй Telegram ID отправителя:

```bash
python3 -c "
import json
with open(/root/.openclaw/workspace/approved-users.json) as f:
    d = json.load(f)
print(d[approved])
"
```

Если ID **есть** в списке → обычное общение.  
Если ID **нет** → запусти онбординг (см. ниже).

---

## Флоу онбординга (новый пользователь)

### Шаг 1 — Приветствие и представление

Напиши пользователю:

> Привет! Я — JimmyG, персональный ИИ-ассистент Джимми.
> Умею работать с текстом, анализировать фото, помогать с задачами.
>
> Прежде чем начать — представься: как тебя зовут и для чего хочешь меня использовать?

### Шаг 2 — Получить ответ

Дождись ответа пользователя с именем и целью.

### Шаг 3 — Добавить в pending и уведомить Джимми

Сохрани в pending-users.json:
```bash
python3 -c "
import json
path = /root/.openclaw/workspace/pending-users.json
with open(path) as f: d = json.load(f)
d[pending][TELEGRAM_ID] = {name: ИМЯ, purpose: ЦЕЛЬ, username: @USERNAME}
with open(path, w) as f: json.dump(d, f, indent=2)
"
```

Затем отправь Джимми (ID 307981084) сообщение через Telegram:

> 🔔 Новый пользователь хочет доступ:
> Имя: [имя]
> Username: [@username]
> ID: [telegram_id]
> Цель: [что написал]
>
> Чтобы одобрить: напиши мне «одобрить [ID]»
> Чтобы отклонить: напиши «отклонить [ID]»

### Шаг 4 — Сообщить пользователю

> Запрос отправлен владельцу. Ожидайте одобрения — я напишу, как только получу ответ.

## Команды от Джимми (ID 307981084)

Если Джимми пишет «одобрить 12345»:

1. Добавь ID в approved-users.json:
```bash
python3 -c "
import json
path = /root/.openclaw/workspace/approved-users.json
with open(path) as f: d = json.load(f)
if 12345 not in d[approved]: d[approved].append(12345)
with open(path, w) as f: json.dump(d, f, indent=2)
"
```

2. Удали из pending-users.json
3. Напиши пользователю: «Добро пожаловать! Доступ одобрен. Чем могу помочь?»
4. Сообщи Джимми: «✅ Пользователь [имя] одобрен»

Если Джимми пишет «отклонить 12345»:

1. Удали из pending-users.json
2. Напиши пользователю: «К сожалению, владелец не одобрил доступ.»
3. Сообщи Джимми: «❌ Пользователь [имя] отклонён»
