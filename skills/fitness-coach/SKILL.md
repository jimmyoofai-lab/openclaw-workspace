# Fitness Coach - Личный тренер

Персональный тренер для фитнеса. Создает планы, отслеживает прогресс, мотивирует.

## Быстрый старт

```bash
# Первая тренировка - создание профиля
чад fitness-coach profile

# Получить план на неделю
чад fitness-coach plan

# Логировать результат тренировки
чад fitness-coach log --exercise "отжимания" --reps 30 --sets 3

# Статистика прогресса
чад fitness-coach stats

# Мотивационный толчок
чад fitness-coach motivate
```

## Как работает

### Профиль
- Твой уровень (начинающий, средний, продвинутый)
- Цели (похудение, масса, выносливость, просто здоровье)
- Доступное время (30мин, 1час, полтора часа)
- Предпочтения (дома, зал, пробежка)
- История (что уже тренировал, результаты)

### План тренировки
- **Структурированный** — разминка, основное, заминка
- **Прогрессивный** — усложняется каждую неделю
- **Адаптивный** — учитывает твой прогресс и время
- **Мотивирующий** — показывает результаты и улучшения

### Отслеживание

## Команды

### Профиль
```bash
чад fitness-coach profile              # Показать профиль
чад fitness-coach profile update       # Обновить (новые цели, уровень и т.д.)
```

### Тренировка
```bash
чад fitness-coach plan                 # План на неделю
чад fitness-coach plan --focus legs    # План конкретной части тела
чад fitness-coach today                # Тренировка на сегодня
```

### Логирование
```bash
чад fitness-coach log --exercise "отжимания" --reps 25 --sets 3
чад fitness-coach log --exercise "бег" --distance 5km --time 30min
чад fitness-coach log --exercise "приседания" --reps 50 --sets 4
```

### Статистика
```bash
чад fitness-coach stats                # Все показатели
чад fitness-coach stats --week         # За неделю
чад fitness-coach stats --month        # За месяц
чад fitness-coach streak               # Твоя текущая серия тренировок
```

### Мотивация
```bash
чад fitness-coach motivate             # Мотивирующее сообщение
чад fitness-coach goal status          # Прогресс к целям
```

## Примеры

### День 1 - Создание профиля
```
Чад, создай мне профиль
→ Какой уровень подготовки? (начинающий / средний / продвинутый)
→ Цели? (похудение / масса / выносливость / здоровье)
→ Сколько времени в день? (30 / 45 / 60 / 90 минут)
→ Где тренируешься? (дома / зал / улица)
```

### День 2 - Первая тренировка
```
Чад, дай мне план на неделю
→ Вот твоя программа (пн-вс)
  Пн: Верх (отжимания, подтягивания, спина)
  Вт: Низ (приседания, выпады, икры)
  Ср: Кардио (бег 30мин, скакалка)
  ...
```

### День 3 - Логирование
```
Чад, я сделал отжимания 25 раз 3 подхода
→ Отлично! Логирую. Прогресс: неделю назад было 20 раз. +25%! 💪

Чад, статистика
→ 3 дня подряд ✅
  Лучший результат: отжимания (25 раз)
  Осталось 27 дней до твоей цели
```

## Интеграция

### Apple Reminders
```bash
# Автоматически создает напоминания о тренировках
чад fitness-coach reminders on
# Пн, Ср, Пт в 18:00 - тренировка ✅
```

### Things 3
```bash
# Добавляет тренировки в список дел
чад fitness-coach sync-things
```

## Мотивация

Тренер знает что:
- Ты сова (тренировки лучше вечером)
- Работаешь в морге (нужна физическая активность)
- Хочешь удаленку (здоровье = продуктивность)
- Любишь результаты (покажу прогресс числами)

Будет пинать за пропуски, праздновать достижения, и не давать сдаться.
